Appendix: How to use Gitbook

因为Gitbook在生成的时候会清空_book这个文件夹，所以我建议

把如下代码保存为 generate.sh，每次编辑后运行即可

gitbook build
cp _book/* Future-User-Interface
cd Future-User-Interface
git add .
git commit -m "update"
git push origin gh-pages
